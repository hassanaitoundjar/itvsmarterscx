Xtream Codes API is a popular protocol used by many IPTV service providers to deliver streaming content. IPTV Smarters Pro fully supports Xtream Codes API, making it easy to connect to your IPTV service. This guide will walk you through the login process step by step.

## What is Xtream Codes API?

Xtream Codes API is a streaming platform that allows IPTV service providers to deliver content to their subscribers. It provides:

- Live TV channels
- Video on Demand (VOD) movies
- TV series and shows
- Electronic Program Guide (EPG)
- Catch-up TV functionality

## Prerequisites

Before you begin, make sure you have:

- IPTV Smarters Pro installed on your device
- An active IPTV subscription that supports Xtream Codes API
- Your Xtream Codes API credentials from your IPTV provider:
  - Server URL
  - Username
  - Password
- A stable internet connection

## Step-by-Step Login Guide

### Step 1: Launch IPTV Smarters Pro

1. Open the IPTV Smarters Pro application on your device
2. You'll see the main screen with options to add a user or login

### Step 2: Add User/Login

1. Tap on **"Add User"** or **"Login"** button
2. You'll be presented with different login options
3. Select **"Login with Xtream Codes API"** or **"Add Xtream Codes API"**

### Step 3: Enter Your Credentials

You'll need to enter the following information:

#### Server URL
- This is typically provided by your IPTV service provider
- Format: `http://your-server.com:port` or `https://your-server.com:port`
- Example: `http://example.iptvprovider.com:8080`
- **Important**: Make sure to include the protocol (http:// or https://) and port number if required

#### Username
- Your IPTV service username
- This is usually provided when you subscribe to the service
- Case-sensitive, so enter it exactly as provided

#### Password
- Your IPTV service password
- Also case-sensitive
- Keep this secure and don't share it with others

### Step 4: Save and Connect

1. After entering all credentials, click **"Add User"** or **"Login"**
2. The app will attempt to connect to your IPTV service
3. If successful, you'll see a success message and be redirected to the main screen

### Step 5: Access Your Content

Once logged in, you can access:

- **Live TV**: Browse and watch live TV channels
- **Movies**: Access your VOD movie library
- **Series**: Watch TV series and shows
- **EPG**: View program schedules
- **Catch-up**: Watch previously aired programs (if supported)

## Troubleshooting Login Issues

### Invalid Credentials Error

**Problem**: App shows "Invalid username or password" error

**Solutions**:
- Double-check your username and password for typos
- Ensure credentials are case-sensitive and entered correctly
- Verify your subscription is active with your IPTV provider
- Contact your IPTV service provider to confirm credentials

### Connection Timeout

**Problem**: App can't connect to the server

**Solutions**:
- Check your internet connection
- Verify the Server URL is correct (including http:// or https://)
- Ensure the port number is included if required
- Check if your IPTV provider's server is online
- Try using a VPN if there are geographic restrictions
- Disable firewall or antivirus temporarily to test

### Server Not Found

**Problem**: "Server not found" or "Unable to reach server" error

**Solutions**:
- Verify the Server URL format is correct
- Check if the server address has changed (contact your provider)
- Ensure you have a stable internet connection
- Try accessing the server URL in a web browser
- Contact your IPTV provider for server status

### App Crashes After Login

**Problem**: App crashes or freezes after entering credentials

**Solutions**:
- Update IPTV Smarters Pro to the latest version
- Clear app cache and data
- Restart your device
- Reinstall the app
- Check device compatibility and system requirements
