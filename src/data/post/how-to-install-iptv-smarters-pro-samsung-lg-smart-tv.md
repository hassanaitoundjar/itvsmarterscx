IPTV Smarters Pro is a powerful streaming application that allows you to access live TV channels, movies, and series on your Smart TV. This guide will walk you through the installation and setup process for both Samsung Smart TV and LG Smart TV.

## Prerequisites

Before you begin, ensure you have:

- A Samsung Smart TV (Tizen OS) or LG Smart TV (webOS)
- An active IPTV subscription with Xtream Codes API credentials
- A stable internet connection
- Your IPTV service provider's server URL, username, and password

## Installation on Samsung Smart TV (Tizen OS)

### Step 1: Enable Developer Mode

1. Press the **Home** button on your Samsung remote
2. Navigate to **Settings** > **System** > **Developer Mode**
3. Enable **Developer Mode**
4. Note down the IP address displayed on your TV

### Step 2: Install IPTV Smarters Pro

1. On your computer, download the IPTV Smarters Pro Tizen app file (.wgt format)
2. Open Samsung Tizen Studio or use the online Tizen package manager
3. Connect to your TV using the IP address from Step 1
4. Install the IPTV Smarters Pro application on your TV

### Step 3: Launch and Configure

1. Open IPTV Smarters Pro from your apps
2. Select **Add User** or **Login**
3. Enter your Xtream Codes API credentials:
   - Server URL
   - Username
   - Password
4. Click **Add User** to save your credentials

## Installation on LG Smart TV (webOS)

### Step 1: Enable Developer Mode

1. Press the **Home** button on your LG remote
2. Go to **Settings** > **General** > **About This TV**
3. Find your TV's IP address
4. On your computer, visit the LG Developer website and register your TV's IP

### Step 2: Install IPTV Smarters Pro

1. Download the IPTV Smarters Pro webOS app file (.ipk format)
2. Use LG webOS TV Developer Mode or LG Content Store Developer Mode
3. Install the application using the developer tools
4. The app will appear in your LG Smart TV apps list

### Step 3: Setup and Login

1. Launch IPTV Smarters Pro from your apps
2. Click **Add User** to add your IPTV service
3. Enter your Xtream Codes API information:
   - Server URL
   - Username
   - Password
4. Save your settings and start streaming

## Using IPTV Smarters Pro

Once installed and configured, you can:

- **Browse Live TV Channels**: Access all available live TV channels from your subscription
- **Watch Movies**: Browse and watch movies from your IPTV service's movie library
- **Watch Series**: Access TV series and binge-watch your favorite shows
- **Use EPG (Electronic Program Guide)**: View program schedules and information
- **Catch-up TV**: Watch previously aired programs if supported by your service

## Troubleshooting

### Connection Issues

- Verify your internet connection is stable
- Double-check your server URL, username, and password
- Ensure your IPTV subscription is active
- Try restarting the application

### Playback Issues

- Check your internet bandwidth (minimum 10 Mbps recommended for HD)
- Clear the app cache and restart
- Update to the latest version of IPTV Smarters Pro
- Contact your IPTV service provider for server status

### App Not Loading

- Ensure Developer Mode is properly enabled
- Reinstall the application
- Check if your TV's firmware is up to date
- Verify compatibility with your TV model

## Tips for Best Experience

1. **Use Ethernet Connection**: For stable streaming, use a wired ethernet connection instead of Wi-Fi
2. **Close Other Apps**: Close unnecessary apps to free up memory
3. **Regular Updates**: Keep IPTV Smarters Pro updated to the latest version
4. **Quality Settings**: Adjust video quality based on your internet speed
5. **EPG Refresh**: Regularly refresh your EPG for accurate program information

IPTV Smarters Pro offers a seamless streaming experience on Samsung and LG Smart TVs. By following this guide, you can easily install and configure the app to enjoy your favorite IPTV content on the big screen.
